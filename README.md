# Slab & Grain Flooring

Premium flooring installation for Tucson, AZ. Hardwood, tile, luxury vinyl, epoxy, polished concrete, and floor repair.

**Live:** https://slab-and-grain-flooring.vercel.app

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Supabase (leads DB)
- Resend (email notifications)
